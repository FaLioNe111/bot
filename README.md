# Ora — Telegram Productivity Bot

Персональный помощник по продуктивности для Telegram.

## Возможности

- 📅 Календарь событий с поддержкой естественного языка и повторяющихся встреч
- 📝 Заметки (текст, чеклисты с авто-определением, структурированные)
- ⏰ Напоминания с гибкими форматами ("через 30 минут", "завтра в 9")
- ✅ Задачи и трекер привычек со статистикой и стриками
- 📊 Утренний брифинг (Pro) с советами дня
- 🔍 Поиск по всем данным (заметки, события)
- 📤 Экспорт данных (Pro)
- 💳 Freemium модель с мягким апгрейдом

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```

2. Получите токен бота у [@BotFather](https://t.me/BotFather) и добавьте в `.env`:
   ```
   BOT_TOKEN=your_bot_token_here
   ```

3. (Опционально) Настройте цену Pro в Stars:
   ```
   PRO_PRICE_STARS=150
   ```

## Запуск

```bash
python bot/main.py
```

Бот запустится в режиме polling и будет обрабатывать сообщения.

## Примеры использования

**Создать событие (естественный язык):**
```
встреча с Алексом в пятницу в 15:00
созвон завтра в 10
каждый понедельник в 9 — планёрка
```

**Напоминание:**
```
напомни купить хлеб через 2 часа
напомни за 20 минут до встречи
каждый воскресный вечер в 20:00 — подготовиться к неделе
```

**Заметка (авто-чеклист):**
```
запиши идеи для проекта:
- Добавить аналитику
- Пересмотреть онбординг
- Провести ревью дизайна
```

**Привычка:**
```
хочу отслеживать привычку — читать 20 минут каждый день
напоминай пить воду каждый час
```

**Поиск:**
```
найди заметки про отпуск
покажи события с врачом
```

## Интерфейс

### Главное меню (Reply Keyboard)
```
[📅 Сегодня] [📆 Неделя] [📝 Заметки]
[⏰ Напомнить] [✅ Задачи] [⚙️ Настройки]
```

### Inline кнопки (контекстные)
После создания события:
```
[✏️ Изменить] [🗑️ Удалить]
[⏰ Добавить напоминание] [📝 Прикрепить заметку]
[✅ Готово]
```

При срабатывании напоминания:
```
[✅ Выполнено] [⏰ Позже: 15 мин]
[⏰ Позже: 1 час] [⏰ Завтра]
```

## Монетизация

### Free план
- До 50 событий/мес
- До 30 заметок
- До 15 напоминаний
- До 3 привычек
- До 5 повторяющихся событий
- Нативная реклама (1 раз в день в брифинге)

### Pro план (~2$/мес или 150 Stars)
- Всё без ограничений ✅
- Утренний брифинг с персональными советами
- Полная статистика привычек (месяц, год, экспорт)
- Расширенный поиск (по тегам, тексту, датам)
- Умный анализ свободного времени
- Шаблоны событий и заметок
- Интеграция с Google Calendar / iCal (webhook)
- Без рекламы
- Приоритетная поддержка

### Апгрейд
- Триал: 7 дней бесплатно
- Реферальная программа: +7 дней за друга
- Telegram Stars для оплаты

## Архитектура

1. **bot/main.py**: Aiogram dispatcher, обработчики команд, callback, polling
2. **services/storage.py**: In-memory хранилище с индексами по пользователям
3. **services/core_services.py**: Бизнес-логика (CalendarService, NoteService, ReminderService, HabitService)
4. **utils/date_parser.py**: Парсинг естественного языка (даты, время, длительность, повторения)
5. **utils/formatters.py**: Форматирование вывода (события, расписания, кнопки)
6. **models/data_models.py**: Pydantic модели для валидации данных
