# Telegram Bot для Алтайского Государственного Университета

## Описание

Этот Telegram-бот позволяет студентам Алтайского Государственного Университета получать информацию из личного кабинета университета, включая:

- Расписание занятий
- Академические задолженности
- Отчеты для классного руководителя
- Отчеты для преподавателей

## Функциональность

- `/start` - Приветственное сообщение и список команд
- `/login <логин> <пароль>` - Авторизация в системе университета
- `/schedule` - Просмотр расписания на 7 дней вперед
- `/debt` - Проверка академических задолженностей
- `/report_class_teacher` - Формирование отчета для классного руководителя
- `/report_lecturer` - Формирование отчета для преподавателя

## Архитектура

Проект состоит из следующих компонентов:

- `main.go` - Основная точка входа в приложение
- `database/` - Работа с SQLite базой данных
- `handlers/` - Обработчики команд бота
- `parsers/` - Парсеры данных из личного кабинета университета
- `utils/` - Вспомогательные функции, включая безопасность
- `models/` - Модели данных

## Безопасность

- Валидация пользовательского ввода
- Защита от SQL-инъекций
- Логирование подозрительной активности
- Ограничение частоты запросов
- Блокировка пользователей при подозрительной активности

## Установка и запуск

1. Убедитесь, что у вас установлен Go версии 1.21 или выше
2. Установите зависимости: `go mod tidy`
3. Установите необходимые системные зависимости для SQLite: `sudo apt-get install gcc libsqlite3-dev`
4. Установите переменную окружения `BOT_TOKEN` с токеном вашего Telegram-бота
5. Запустите приложение: `go run main.go`

## Конфигурация

Для работы бота требуется токен Telegram-бота, который должен быть установлен в переменной окружения `BOT_TOKEN`.

## База данных

Проект использует SQLite для хранения:
- Информации о пользователях
- Сессий авторизации
- Расписания занятий
- Академических задолженностей
- Отчетов
- Логов безопасности

Схема базы данных находится в файле `database/schema.sql`.

## Структура проекта

```
altstu_bot/
├── main.go                 # Основная точка входа
├── go.mod                  # Зависимости модуля Go
├── go.sum                  # Контрольные суммы зависимостей
├── database/               # Работа с базой данных
│   ├── db.go              # Подключение и миграции
│   └── schema.sql         # Схема базы данных
├── handlers/              # Обработчики команд бота
│   └── user_handlers.go   # Обработка пользовательских команд
├── parsers/               # Парсеры данных из личного кабинета
│   └── parser.go          # Основной парсер
├── utils/                 # Вспомогательные функции
│   └── security.go        # Функции безопасности
└── models/                # Модели данных
```

## Замечание по парсеру

Файл `parsers/parser.go` содержит заглушку для парсинга данных из личного кабинета университета. Реализация должна быть дополнена в соответствии со структурой сайта Алтайского Государственного Университета. Для этого потребуется:

1. Изучить структуру сайта личного кабинета
2. Реализовать аутентификацию
3. Добавить парсеры для расписания, задолженностей и других данных
4. Обеспечить надежную обработку ошибок и безопасность
